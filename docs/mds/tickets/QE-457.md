# QE-457 — SPA Vintage Inspector screen (mirrors evolve PoolReview)  *(R1 — inspection)*

> **Status:** Todo — [design QE-455](../../architecture/qe-455-research-flow-design.md#72-the-vintage-inspector-screen-qe-457--mirrors-poolreview) · [backlog index](../../backlog.md#research-flow) · [spec](../../specs.md)

`Phase: Research-flow (R1)` · `Area: frontend` · `Depends on: QE-456, QE-259`

**Why.** The SPA still renders a placeholder for the vintage/strategies browser (*"The strategies browser is on
the way"*, `web/src/app/App.tsx:121`), while `evolve` has a full inspection screen
(`web/src/app/evolve/PoolReview.tsx`). Now that QE-456 exposes the composition + gate evidence, give the
operator the equivalent inspection for a **train**-sealed vintage.

**Scope / requirements.**
- A Vintage Inspector screen consuming `GET /api/vintages/{id}` (QE-456), reachable from the vintages list
  (replacing the `App.tsx:121` placeholder), structured like `PoolReview.tsx`:
  - **Composition table** — chromosome → referenced indicators (catalogue vs evolved, labelled) → weight.
  - **Selected-indicators panel** — the union of indicators the ensemble references.
  - **Gate-evidence card** — leads with the **net-of-cost / tradability** numbers (cost-stress `min{1×,2×}`,
    turnover, capacity) and the **honest deflation basis** (distinct-trial N vs the `E[maxSharpe]` bar,
    uncensored PBO with its population size, **DSR labelled "necessary — not sufficient"**) — the same hierarchy
    QE-450 §13.5 mandates for PoolReview. Sharpe/DSR demoted, never a lone green tile.
  - **Frozen-holdout panel** — the recorded `{train_range, embargo, holdout_range}` split (design §4).
- Read-only: **no seal / promote / select affordance** (the vintage is already sealed by the train gate).

**Out of scope.** The endpoint (QE-456). The leaderboard/comparison view (QE-466). Steering controls (QE-459).
Any client-side gate recomputation (render server-provided numbers verbatim).

**Acceptance criteria.**
- The screen renders composition (chromosome→indicators→weight), selected indicators, the gate/deflation
  evidence, and the holdout split from QE-456's payload; the placeholder is removed.
- The gate card leads with net-of-cost/tradability and shows the deflation basis honestly (N vs bar, uncensored
  PBO + population, DSR marked necessary-not-sufficient); it presents no standalone health badge.
- No promote/seal/select control exists anywhere on the screen (inspection only).
- Component tests cover the composition table + gate card, mirroring `PoolReview.test.tsx`.

`Spec ref: QE-455 §7.2 "Vintage Inspector screen"; QE-450 §13.5 (the minimum honest stat set, hierarchy).`
