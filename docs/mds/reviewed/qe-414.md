## Review verdict — [Approved]
`cell_champion_returns` gathers the best elite of every occupied cell (both directions), size = `archive.occupied_cells()` — exactly the cell factor in `n_trials = occupied_cells × generations × windows`, so `trial_variance` and `n_trials` derive from the same population. MAP-Elites retains only elites-per-cell (no raw per-evaluation Sharpes), so cell champions are the maximally-uncensored sample the archive exposes; the spec permits "a representative uncensored sample." `lib.rs` cleanly separates `variance_returns` (DSR dispersion, champions) from `trial_returns` (CSCV/PBO/SPA, top-N pool) — scope contained, ensemble/CSCV/SPA unchanged. Regression test non-vacuous (asserts strict `dsr_full < dsr_top10` on a spreading-Sharpe 20-cell population). No golden/G1 flip (DSR not embedded in sealed content; sealed hash byte-identical; the seal test now also asserts `variance_trials(38) > pool_size(10)`). Wire `Gate` line deliberately unchanged (preserves QE-406 agreement; `dsr` already on the wire). Determinism preserved (fixed-order gather, no RNG).

Non-blocking nit (recorded, conditional follow-up — not ticketed): cell champions still capture one Sharpe per niche, not within-cell dispersion, so the estimator remains partially censored and DSR could stay mildly inflated vs a true all-evaluations population. Fully closing it requires MAP-Elites to retain per-evaluation Sharpes (out of scope). Worth a tracked follow-up only if DSR calibration is later found wanting.
