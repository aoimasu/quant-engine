## Decision summary

Each occupied `Cell` is a Normal–Normal bandit arm over its mean **in-training** reward
(`NichePrior {mean, var, obs_var}`). `select_parent` Thompson-samples each occupied niche (Box–Muller
from the seeded `DetRng`), argmaxes, then samples an elite uniformly within → `(Cell, &Genome)`.
`record(cell, &ApplicationOutcome)` is the ONLY posterior-update path — the reward is the QE-112/119
in-training novelty/improvement credit; there is no field or parameter for a held-out/validation score
(QE-001/QE-132 firewall). Variance floors at `MIN_VARIANCE = 1e-12` so a near-zero-variance prior cannot
overflow precision to NaN.

## Review — Approved (passes 1 & 2)

**Pass 1** (`bfae936`): no-OOS-leakage AC structurally true and non-vacuously tested (in-training vs a
never-passed validation signal made anti-correlated; selection follows in-training A not validation B,
`a > 5b`). Normal–Normal math correct (`precision = 1/σ0² + n/σ²`, `mean = (μ0/σ0² + s/σ²)/precision`,
`var = 1/precision`); one posterior draw per arm → argmax → uniform elite is real Thompson, not
argmax-of-mean. Deterministic (single `DetRng`, sorted `BTreeMap` order, pinned by `count_a(7)==count_a(7)`).
Concentrates on high-reward niches yet still explores unseen ones (prior variance). Edges safe (empty →
None, single-cell, `ln(0)` floor, negligible modulo bias).

Two non-blocking notes: (1) the `f64::MIN_POSITIVE` variance floor was a NaN footgun for a near-zero
caller variance; (2) fixed `obs_var` vs a per-niche estimate (deferred).

**Pass 2** (`75e2b02`): applied note 1 — `MIN_VARIANCE = 1e-12` floor (`1/1e-12 = 1e12`, well within f64,
so precision/mean/var stay finite for any input) + a `degenerate_prior_variance_stays_finite` guard test.
`git diff bfae936..75e2b02` touches only `thompson.rs` (+26/−2); the firewall, conjugate math,
determinism, and leakage AC are byte-for-byte untouched, so the pass-1 approval stands. Note 2 deferred.
