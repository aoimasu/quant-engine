### Goal
Emit absolute target positions; stateless wrt current position; track equity and buying power.

### Acceptance criteria (from backlog)
- [x] The planner emits identical targets regardless of current venue position (statelessness) —
  `plan_is_stateless_wrt_current_venue_position`.
- [x] Equity/margin view matches keeper truth — `capital_view_matches_keeper_truth`.

### Implementation summary
- `PositionKeeper` seam → `CapitalView { equity, available_margin }` + `venue_position()` + default `equity()`
  (keeper truth; real keeper = QE-217, fake in tests). `HedgePlanner::plan(net)` → absolute `TargetPosition {
  notional }` = `net.net × equity` (fresh read via `keeper.equity()`); **never reads `venue_position`** →
  stateless (absolute target, not a delta; `target − current` is QE-217). `capital_view()` forwards keeper
  truth. `Notional` signed (sign = direction; `direction()` decodes).
