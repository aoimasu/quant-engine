## Decision summary

Three operators (mutate-freely-then-repair): `local_refine` (exploitation, **cell-preserving** ‚Äî nudges
bounds + size only), `explore` (exploration, cell-changing ‚Äî re-points a feature + resets holding),
`fresh_random` (maximal exploration). `VariationDriver::step` ties select ‚Üí sample parent ‚Üí vary ‚Üí eval ‚Üí
insert ‚Üí credit, mapping QE-118 `InsertOutcome` ‚Üí QE-112 `ApplicationOutcome` (`NewCell`‚Üínovelty,
`ImprovedElite`‚Üínormalised displaced `gain`, `Added`/`Rejected`‚Üíno credit). The budget shift is
**emergent**: sparse archive ‚áí jumpers fill empty cells (credit) while `local_refine` only `Added`s;
dense archive ‚áí jumpers `Rejected` while `local_refine` improves an elite. Two additive QE-118 accessors
(`SubPopulation::worst`, `MapElitesArchive::sample_parent_elite`) supply the parent fitness for the gain.
