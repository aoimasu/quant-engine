## Decision summary

`assess_robustness` runs three gates over an elite and a ±ε jitter cloud (each sample seeded by
`task_rng(seed, i)`) and returns a flagging `RobustnessReport`: (1) minimum-trade-count, (2)
parameter-perturbation robustness (collapse = non-finite / below trade floor / retains `< retain_fraction`
of base fitness; gate fails when `collapsed_fraction > max_collapse_fraction`), (3) descriptor stability
via QE-111 `cell_reassignment_rate` (≤ `STABILITY_THRESHOLD`). `jitter` nudges only parameter genes (clause
`lo`/`hi`, `max_holding_bars`, `size_bps`) then `repair`s; structural genes untouched. It flags, the caller
(QE-123/QE-125) decides reject-vs-quarantine.

## Review (pass 1) — Approved

Principal-level review of `8df8c04`. Both halves of the AC are demonstrated by genuine, mechanistic
fixtures (not gamed); jitter/collapse/descriptor logic sound; assessment deterministic. No real bugs.

- **(a) Fixtures genuine.** Fragile: the `{3}`-band elite enters only on state-3 rising legs; `min_trades=1`
  rules out a no-trade artefact; ±1 jitter widens the band to catch falling-leg TRAP trades → fitness
  drops below `0.5·base` → `collapsed_fraction ≫ 0.25` → `perturbation_ok = false`; a no-op gate would give
  `collapsed = 0` and fail the assertion. Boundary: `max_holding_bars = 6` flips Scalp→Swing under ±2
  holding jitter (~0.4 rate); `descriptor_for(Long)` is genuinely `Some` (feature 0 classifiable), so real
  `Some↔Some` reassignments are counted, not vacuous `None`.
- **(b) Collapse definition** sound for the intended `base > 0` domain (retain ≥ 50%, ruin/non-finite
  collapses, improvement never collapses).
- **(c) Jitter** correct uniform `[-eps,eps]` + saturate + repair; not touching `enabled`/`feature` is the
  right scope — family/timescale stay fixed so holding is the only axis a parameter nudge can flip, a
  complete probe.
- **(d) Descriptor gate** handles one/both/neither occupied directions; neither ⇒ rate 0 ⇒ acceptable.
- **(e) Deterministic** via `task_rng(seed, i)` per sample; pinned.
- **(f) Edges:** `samples=0`/`eps=0`/empty bars handled without panic.

Two non-blocking forward notes, no action required: (1) the `fit < retain_fraction·base_fitness` rule
inverts for `base ≤ 0` (benign — unprofitable elites are filtered upstream; a documented precondition would
be cleaner, and is captured in the design-note Risks); (2) the perturbation/descriptor gates are only as
strong as `samples > 0` / `eps > 0` (the defaults are sound).

No outstanding comments. Approved & merged.
