### Goal
Per-bar decisions net into a single aggregate target position. Net per-strategy (post-breaker) decisions into
one aggregate target per instrument.

### Acceptance criteria (from backlog)
- [x] Netting equals the sum of post-breaker per-strategy targets; gated strategies contribute zero —
  `net_equals_sum_of_leg_targets`, `flat_leg_contributes_zero`, `gated_strategy_via_breaker_contributes_zero`.

### Implementation summary
- `NetLeg { direction, weight: Decimal, size_bps }`; `signed_target()` = `weight × size_bps/10_000` signed by
  direction (`0` when flat). `PositionNetter::net` → `NetTarget { net = long − short, long, short }` (+
  `gross()`); `net_positions` zips post-breaker positions/weights/sizes (hard `assert!` on alignment). f64
  weight → Decimal once via `from_f64_retain` (`debug_assert` finite).
- **Gated → zero by construction**: QE-212 `clamp` → `Exit` → `advance` → flat → target `0` (proven
  end-to-end vs `BreakerLayer`). Aggregate is a fraction of capital (absolute scaling = QE-214);
  `long`/`short` split advances the QE-212 per-direction forward obligation + QE-215 caps.
