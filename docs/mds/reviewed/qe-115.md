## Decision summary

Lives in `qe-ensemble`, **independent of `qe-wfo`** (search ⟂ portfolio firewall, QE-001/QE-132 —
self-contained `f64`). Objective = `mean + tail_weight·CVaR − corr_weight·positive_mean_pairwise_corr`
on net-of-cost combined returns (D3+D5). `cvar`/`cdar` return `tail_n` (the SE caveat made explicit, D3);
`stress_overlay` augments empirical tails with synthetic shocks (D4); the correlation penalty floors
negative correlation to 0 (D5); `leave_one_out_min` is the wide-basin floor (D6). Discrete-DE operators
(D1/D2): `EnsembleMask` binary representation, `de_mutant = a XOR (b XOR c)`, seeded deterministic
`binomial_crossover` with a guaranteed `j_rand` locus. The DE loop is QE-126.
