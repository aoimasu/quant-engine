3. AC (c): `enforce_coverage_floor` called live, hard-errors below `MIN_OCCUPIED_NICHES` (`coverage_floor_is_enforced_only_on_steered_runs`).
4. Item 4: `indicator_subset` applied as a leak-free RNG-neutral feature allowlist in `qe_wfo::variation` + `windows`/`folds` override, threaded via `spawn.rs`; unknown indicator → hard `RunError::UnknownIndicator` (no silent no-op). `empty_allowlist_is_byte_identical_to_the_unmasked_search`.
