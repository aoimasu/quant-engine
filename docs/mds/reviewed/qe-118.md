## Decision summary

`MapElitesArchive` = two per-direction `BTreeMap<Cell, SubPopulation>` over the QE-111 niche grid; a
genome is placed at the cell its that-direction bank describes. `SubPopulation` is bounded to
`SUBPOP_SIZE` (Deep-Grid); `consider` → `NewCell`/`Added`/`ImprovedElite` (strict-better replacement of
the lowest-index worst) / `Rejected` — the outcome hook QE-119 reads. `sample_parent` is
uniform-cell-then-uniform-elite (Deep-Grid diversity). `evaluate_batch` is the determinism lever:
`rayon::par_iter` where each task seeds its own `DetRng` from `task_rng(master,index)` (no shared RNG) +
ordered `collect` → byte-identical regardless of pool size.

## Review (pass 1) — Approved

Principal-level review of `86ec021`. Both ACs genuinely met — the determinism AC is real, not vacuous —
sub-populations correctly bounded, no reachable panics or determinism holes. No real bugs.

- **(a)** `evaluate_batch` gives each task its own `task_rng(master,index)` stream (ChaCha8, QE-006), no
  shared RNG/accumulator, ordered `collect` → byte-identical for any pool size; the 1-vs-8-thread test is
  non-vacuous (`rng_eval` consumes the rng, so a shared-rng impl would diverge). `evaluate_and_insert`
  inserts sequentially in input order → pool-size-independent archives.
- **(b)** `len` never exceeds `SUBPOP_SIZE`; full cells replace the lowest-index worst only on strict `>`
  (no churn); converges to top-`SUBPOP_SIZE`; `total_cmp` throughout (no NaN footgun).
- **(c)** Long/Short descriptors computed independently and placed per-bank; `sample_parent` uniform over
  the sorted `BTreeMap` reaches sparse cells, seed-deterministic; no `HashMap` anywhere.
- **(d)** Modulo bias ~`2⁻⁵⁹` for `len ≤ 45`, negligible.
- **(e)** The sole `expect` is unreachable (full branch ⇒ non-empty cell); `sample_parent` uses guards +
  `?`/`.get()`. No panic on a valid archive.

Two non-blocking forward notes, no action: (1) the `consider` `expect` would only become live if
`SUBPOP_SIZE` were ever set to 0 — a `debug_assert` worth adding only if that constant becomes
configurable; (2) the negligible modulo bias in parent sampling.

No outstanding comments. Approved for merge.
