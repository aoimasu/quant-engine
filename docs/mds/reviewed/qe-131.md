## Review (pass 1) — Approved

Principal-level pass; formulae checked against the published methods, not just tests. Verdict:
**Approved.** No real bugs.

- **Firewall intact** — `Cargo.toml` deps `qe-determinism`/`rand_core`/`serde`/`thiserror` only; no
  `qe-wfo`/`qe-ensemble`.
- **(a) DSR (Bailey & LdP)** — PSR `Φ[(SR−SR*)√(T−1)/√(1−γ3·SR+((γ4−1)/4)SR²)]` exact; `kurtosis`
  **non-excess** (m4/m2², normal=3) so `(γ4−1)/4` is right (excess/non-excess trap avoided);
  `expected_max_sharpe` with γ=Euler–Mascheroni matches; `effective_trials` saturating; guards correct.
- **(b) PBO/CSCV** — `logit ≤ 0 ⇔ rank ≤ (N+1)/2 ⇔ IS-best at/below OOS median ⇔ overfit`, exactly Bailey
  et al.; robust matrix → PBO 0 (an inverted impl would fail it), odometer `combinations` complete.
- **(c) SPA/RC (White/Hansen)** — bootstrap statistic is **recentred** (`resampled_mean − d̄ₖ`); stationary
  bootstrap restarts with `q=1/avg_block`, wraps, `task_rng`-seeded (deterministic); studentise divides by
  σₖ; NaN-safe.
- **(d) Nulls** — HODL `pₜ/pₜ₋₁−1` (guarded); random-entry flips with prob `p` ⇒ realised turnover ≈ `p`,
  deterministic per seed.
- **(e) Stats** — A&S erf + Acklam ppf spot-checks; population skew/non-excess-kurtosis/ddof=1 std; `assess`
  serde round-trips (G1-ready) and propagates CSCV errors.

One non-blocking note: the `overfit_matrix_has_high_pbo` fixture used constant-within-block values (Sharpe
dominated by a near-zero-dispersion mean → FP-fragile); direction not in doubt (cleanly pinned by the
robust → PBO 0 test).
