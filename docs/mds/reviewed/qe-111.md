## Decision summary

All three descriptor axes are **genotype-derived** (D1) — `descriptor_for(genome, direction, schema)`
reads no window/outcome data, so the niche is window-invariant by type signature: `IndicatorFamily`
(dominant family of the direction-bank's enabled clauses), `TimescaleBand` (max referenced lookback),
`HoldingBand` (`exit.max_holding_bars`). Grid = 5×3×3 = **45 cells/direction** (D2, coarse,
min-trades-per-cell rationale); `SUBPOP_SIZE = 8` Deep-Grid sub-populations (D3, Flageat & Cully 2020);
per-direction archives so the ensemble is not net-long by construction (D4). Stability metric
`cell_reassignment_rate` with `STABILITY_THRESHOLD = 0.05` (D5): genotype-derived descriptors score
exactly `0.0`, and the metric is proven non-trivial (`2/3` on differing inputs).
