## Review (pass 1) — Approved

Careful principal-level pass on both commits of substance (the genome relocation `2d1d3c6` and the
evaluator session `1cea1d9`), as this is the final ticket before QE-209 and carries an architectural
refactor.

**(A) Genome relocation — a clean, behaviour-preserving move.**
1. _Pure relocation._ `git show 2d1d3c6 --find-renames` reports `crates/wfo/src/genome.rs →
   crates/signal/src/genome.rs` as a rename with only 4 changed lines — all import repoints
   (`qe_signal::{FeatureSchema, FeatureVector}` → `crate::{…}` in the body; `qe_signal::{CatalogueConfig,
   QState}` → `crate::{…}` in the test module). Zero changes to `decide`/`is_valid`/`repair`/`fires` or the
   struct/field definitions.
2. _qe-wfo API + internals unchanged._ `pub mod genome;` became `pub use qe_signal::genome;` with the flat
   re-exports now originating from `qe_signal::genome::{…}`, so `qe_wfo::Genome`, `qe_wfo::genome::*`, and
   the search side's internal `crate::genome::*` resolve exactly as before. 78 qe-wfo tests passing is the
   proof nothing on the training side broke.
3. _Vintage serde / content-hash unchanged — sealed vintages still load._ `qe-vintage` swapped
   `use qe_wfo::Genome` → `use qe_signal::Genome` and dropped `qe-wfo` from its `Cargo.toml`. The `Genome`
   struct is byte-identical and serde encodes field names, not the module path, so the content-hash is
   unchanged and previously-sealed vintages still deserialise/verify (6 qe-vintage round-trip + tamper
   tests pass).
4. _No cycle, deps minimal._ `qe-signal` gained only `serde` (+ `serde_json` dev) — both existing workspace
   deps, deny clean. The `dependency_topology` guard is green: the machine-checked proof that runtime now
   reaches `Genome::decide` through `qe-signal` without any edge to the training side.

**(B) The AC — replay→live continuity with no state copy.**
- `go_live` sets only `self.mode`; the decision computation (`genome.decide` + `advance`) is wholly
  mode-independent, so the decision stream is structurally invariant to the boundary.
- `replay_to_live_transition_does_not_change_decisions` runs all-replay vs switched-at-30 over identical
  vintages and asserts identical per-chromosome decisions bar-for-bar (a reset would diverge post-30);
  `decisions_are_non_trivial` pins Enter+Exit so it isn't vacuous.
- `advance` mirrors the backtest (Enter→held 0, Hold→++, Exit→flat; exit at `max_holding_bars`), verified
  by `single_genome_exits_at_max_holding`. Vintage held read-only (accessors only). `crates/runtime/
  Cargo.toml` has no `qe-wfo` dep.

Non-blocking note: the position `advance` was a small reimplementation that must stay in lockstep with the
backtest's bookkeeping; co-locating it in `qe-signal` next to `decide` would make the pairing a single
source of truth.
