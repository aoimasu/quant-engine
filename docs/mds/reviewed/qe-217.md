### Implementation summary
- `plan_delta(target, current_qty, mark)` → `OrderIntent` (target − kept position, qty space); `None` at
  target/pre-mark. `VenueKeeper` absorbs QE-204 `UserDataEvent` (Fill adjusts; Position/Snapshot SET —
  overrides fills; never infers); `observe_mark`/`observe_balance`; impls the QE-214 `PositionKeeper` seam
  (`equity()` left default → QE-214 consistency obligation satisfied); `&K` blanket impl lets a `HedgePlanner`
  borrow it. `VenueSimulator` = in-memory venue, immediate-fill, emits the Fill event; full loop, no real
  orders. `Order` lifecycle with terminal-state guards (`is_terminal`).
