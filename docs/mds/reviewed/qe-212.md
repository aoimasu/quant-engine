### Goal
Per-strategy/direction/ensemble limits + slow/med/fast DD thresholds clamp gated strategies to flat before
netting. Implement the QE-116 breaker model consuming the calibration profile + equity stream.

### Acceptance criteria (from backlog)
- [x] A breach clamps the affected scope to flat before netting — `strategy_breach_clamps_that_strategy_to_flat`,
  `ensemble_breach_clamps_all_strategies` (gated → `Decision::Exit`).
- [x] Behaviour matches the historical backtest (QE-116) on replay — `live_layer_matches_qe116_replay`
  (layer's `(index, tier)` events equal `qe_risk::replay` on the same equity stream).

### Implementation summary
- `BreakerLayer` wraps the QE-116 `CircuitBreaker`: one per strategy + one ensemble (fast-drop-only, slow/med
  disabled via `never_fires()` = 1.0). `from_calibration` consumes the profile; an uncalibrated strategy is
  **explicitly pre-gated** (a `pre_gated` mask, independent of tier thresholds) and given a fires-immediately
  breaker as defence in depth. `observe_strategy`/`observe_ensemble` latch gated scopes; `clamp` rewrites any
  gated strategy's decision to `Decision::Exit` (flat) before netting; `reset` re-arms breakers and
  **re-applies** the `pre_gated` fail-safe.
- Scope: per-strategy + ensemble implemented; per-direction/cohort deferred to the same mechanism (see
  forward obligation). Equity stream is a declared input (net-of-cost feed = QE-217), the QE-210 boundary.

### Review notes — [Approved] (three passes)

**Pass 1 (@ `cbda05d`).** No blocking defects. AC half 1 (clamp) proven — gated → `Exit` while ungated pass;
`Exit` is the correct clamp-to-flat (position driven & held flat → zero netted contribution). AC half 2
(replay parity) proven exactly and non-vacuously — `observe_strategy` delegates to the same
`CircuitBreaker::observe` with the same `DEFAULT_FAST_WINDOW`+thresholds, so the layer fires tier-for-tier
identically to `qe_risk::replay`; the fixture asserts `!expected.is_empty()`. Latching correct; ensemble
fast-drop-only correct; missing-calibration fires **Med** on tick one (verified against `breaker.rs`: the
zero fast tier is skipped by the `fast_thresh > 0` guard, then `drawdown(0) >= med_dd(0)`). No firewall/parity
concern (reuses the primitive, so live gating can't drift from the backtest).

Non-blocking: (#1) per-direction/cohort deferral is **legitimate** — per-direction gating needs a
per-direction aggregate equity stream produced by netting (QE-213, which *depends on* QE-212 → real
dependency inversion), per-cohort needs the QE-129 map; the mechanism is provably identical; PM-ack requested
since the scope wording names "direction". (#2, subtle) the missing-calibration fail-safe fired only via the
med tier, so a future `>0` guard on med/slow would silently flip it to fail-open — make it explicit. (#3)
`never_fires()` (1.0) actually fires at 100% drawdown — doc it. (#4) `is_gated(out_of_range)` returns
`ensemble_gated` — harmless.

**Pass 2 (@ `cec2c6d`).** #2 addressed structurally — `from_calibration` collects uncalibrated indices and
sets `strategy_gated[i]=true` at construction, gating them independent of the breaker's tier thresholds; the
zero-threshold breaker retained as defence in depth. #3 doc corrected. New non-blocking follow-on: `reset()`
cleared gating without re-applying the pre-gate → a reused layer briefly lost the fail-safe.

**Pass 3 (@ `ff3b1bc`) — confirmed.** Added a `pre_gated` mask (exactly `strategy_ids.len()` long; `reset`'s
`zip` is total); `reset` restores `strategy_gated[i] = pre_gated[i]` instead of `false`, so an uncalibrated
strategy stays gated across a rollover while a tripped-but-calibrated one clears (`ensemble_gated` untouched
by the mask, still clears). New test `reset_reapplies_uncalibrated_fail_safe`. Note #2 hardening complete
end-to-end.
