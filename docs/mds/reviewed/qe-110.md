## Decision summary

Representation = a **fixed-structure parameter vector that encodes a bounded k-of-n band rule set over
quantised feature states** (decision D1) — the hybrid that is the strict superset both downstreams
need: DE/MAP-Elites want fixed, position-wise loci; the spec frames strategies as logic over quantised
states. Per-direction entry banks (`long_entry`/`short_entry`) + fixed `exit`/`risk` genes. `decide()`
is a pure deterministic evaluator (batch/streaming-identical, QE-006); `repair()` is the idempotent
"mutate-freely-then-repair" operator contract (QE-112/119); `referenced_features()` exposes
genotype-derived descriptors for QE-111. `serde` for vintage lineage; `REP_VERSION` header.
