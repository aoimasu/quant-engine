## Decision summary

`search.rs` extracts `pub(crate) run_de(... score closure)` so the base (QE-126) and regime-aware searches
share one deterministic loop (behaviour-preserving). `regime_aware_objective = objective − regime_weight·
max(0, regime_floor − worst_regime_expectancy)`; the QE-115 correlation penalty rides inside `objective`.
`per_regime_expectancy` buckets the ensemble's combined net-of-cost returns by QE-125 regime;
`regime_aware_cv_score` slices both pool and labels per fold. No new deps; firewall intact.
