## Implementation
New crate `qe-clock`: `SkewGuard::evaluate(local, reference) -> SkewReading` (pure, panic-free via
`saturating_sub` + `unsigned_abs`); `ClockHealth { InSync, Skewed }` is the cockpit health signal. A
breach is expressed as QE-004's `QeError::fatal` → `disposition == Halt` (the channel QE-009's kill
switch consumes). `check` = halt-or-continue; `check_and_log` = one-call log-and-halt path;
`record_skew(&reading, &Correlation)` emits a structured `tracing` event (warn on breach) with all
four correlation fields + `skew_ms` + `health`.
