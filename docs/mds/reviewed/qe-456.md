### Endpoint response shape (for QE-457)
`GET /api/vintages/{id}` (session-gated). Top-level: `id`, `label`, `content_hash`, `format_version`,
`data_provenance` (`real|synthetic|mixed`), `composition[]` (`{index, weight, indicators[{feature, id?, source}]}`,
source `catalogue`/`evolved`), `seal_evidence` (dsr/pbo/spa_pvalue/n_trials/realised_turnover/capacity_usd +
optional cost_stress_net_min/uncensored_pbo/ic/fdr), `holdout_series_handle` (64-hex ref, NOT inline) +
`holdout_series_len`, `holdout_split` (holdout_range?/train_range?/embargo_bars), `regime_composition[]`
(`{regime, bars}`), `consultation_count`, `steer_delta?`, `sidecars` (slippage/sizer/worst_case_loss?/calibration/
catalogue), `producing_runs[]` (`{run_id, run_type, status, created_ms}`, deterministic order) + `primary_run?`.
Errors: `404` unknown id; `500` (never panic) on hash/schema/IO failure.
