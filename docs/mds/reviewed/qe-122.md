## Decision summary

`BehaviouralRegulariser { pressure }` weights occupied cells by `1/(1 + pressure·local_crowding)`;
`select_parent_cell` samples proportionally so frontier cells reproduce more and coverage grows along the
frontier. `neighbours` connects the 45-cell behaviour space (±1-band ordinal in timescale/holding + every
other family at the same band — family categorical). `coverage` and `occupancy_entropy` are the diversity
metrics. `pressure = 0` degenerates to uniform (the ablation).

## Review (pass 1) — Approved

Principal-level review of `79e1a71`. The diversity-improvement AC is a genuine mechanistic demonstration
(not gamed); neighbourhood/weight/metric math correct; `pressure = 0` is a true ablation. No real bugs.

- **(a)** Both runs use identical seeds/steps/reachable, only `pressure` differs; `select_parent_cell`
  consumes one rng draw regardless of pressure so both walk the same stream — only the weighted-vs-uniform
  parent mapping differs. Mechanism real (frontier cells reproduce → random neighbour more often empty →
  `NewCell` → +coverage); non-vacuous (strict `>` would fail on no-effect/wrong-sign). `coverage` is the
  standard QD metric; single-feature→representative-cell synthesis is a fair controlled behaviour-space.
- **(b)** `pressure=0 ⇒ weight=1` (uniform), pinned.
- **(c)** `neighbours` = ±1 ordinal timescale/holding (edge-clamped, no self) + every other family at the
  band; counts verified centre 8 / corner 6 / edge 7.
- **(d)** Monotone `1/(1+pressure·crowding)`, correct `local_crowding`, inverse-CDF proportional sampling
  from one `uniform01` draw, deterministic (`run()==run()`).
- **(e)** `occupancy_entropy` correct (0 for ≤1 cell, `ln(2)` for two even cells), guarded.
